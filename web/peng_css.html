Pengenalan Dasar CSS

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan elemen-elemen yang tertulis dalam bahasa markup seperti HTML. Dengan CSS, Anda dapat mengontrol warna, font, tata letak, dan aspek visual lainnya dari halaman web.

Dan CSS digunakan untuk mendeskripsikan bagaimana elemen HTML ditampilkan di layar, kertas, atau media lainnya. CSS juga memisahkan konten (HTML) dari presentasi (gaya visual). Ini berarti Anda dapat mengubah tampilan situs web tanpa mengubah kontennya.

Fungsi Utama CSS

  • Mengontrol Tata Letak: CSS memungkinkan Anda mengatur posisi elemen di halaman web, termasuk margin, padding, dan posisi elemen.

  • Memberikan Gaya Visual: Anda dapat menggunakan CSS untuk mengubah warna, font, ukuran teks, dan aspek visual lainnya dari halaman web.

  • Membuat Responsif: CSS memungkinkan Anda membuat halaman web yang menyesuaikan diri dengan berbagai ukuran layar dan perangkat.

  • Mempermudah Pemeliharaan: Dengan memisahkan gaya dari konten, CSS membuat pemeliharaan situs web lebih mudah. Anda dapat mengubah gaya di satu tempat dan perubahan tersebut akan diterapkan ke seluruh situs.

Cara Kerja CSS

  • CSS bekerja dengan memilih elemen HTML dan menerapkan aturan gaya ke elemen-elemen tersebut. Dan Aturan gaya terdiri dari properti (misalnya, color, font-size) dan nilai (misalnya, red, 16px).

  • CSS dapat ditulis dalam tiga cara:

      Inline: Gaya diterapkan langsung ke elemen HTML menggunakan atribut style.

      Internal: Gaya ditempatkan di dalam tag style di bagian_head dari dokumen _HTML.

      Eksternal: Gaya disimpan dalam file css, terpisah dan dihubungkan ke dokumen HTML menggunakan tag ().

Manfaat CSS

  • Membuat halaman web lebih menarik secara visual.

  • Meningkatkan aksesibilitas dengan menyediakan gaya alternatif untuk pengguna dengan kebutuhan khusus.

  • Mempercepat pengembangan web dengan memungkinkan penggunaan kembali gaya di beberapa halaman.

  • mempermudah dalam perubahan tampilan, tanpa harus merubah struktur HTML.

Konsep Dasar CSS

  • Selektor: Memilih elemen HTML yang akan diberi gaya.

  • Properti: Menentukan aspek gaya yang akan diubah (misalnya, warna, ukuran).

  • Nilai: Menentukan nilai properti (misalnya, merah, 12 piksel).

  • Model Kotak (Box Model): Setiap elemen HTML dianggap sebagai kotak dengan properti seperti margin, padding, dan border.
